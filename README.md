# LinguaMold

A gamified language learning app built with React Native and Expo. 

Built as a passion project and proof of concept for a bigger idea: 
language learning apps built natively for underserved language speakers, 
not as an English-first afterthought.

Learning English --> Spanish is overdone. 
English --> "Obscure" languages as well. 
What about "Obscure" languages --> English/Spanish/Mandarin/etc.

## What is built

12 exercise types (molds): fill-in-blank, translation, word reorder, 
listening, speaking, card match, image select, conjugation, flashcards, 
true/false, conversation, and placement testing.

Full gamification layer: XP, streaks, hearts, gems, achievements, 
weekly leaderboards, and a shop for streak freezes and hearts.

Onboarding with proficiency placement so experienced learners skip 
what they already know.

Adventure path home screen with module nodes, daily progress ring, 
and streak HUD.

Admin panel with full CRUD for modules, lessons, and exercises, 
inline JSON editor, and live exercise preview.

Database-driven localization supporting 9 language columns across 
197 UI string keys.

## Stack

React Native, Expo, Supabase, Zustand, TypeScript

19 Supabase migrations, 2 edge functions, 13 hooks, 50+ components. 
TypeScript strict mode with 0 errors.

## Status

Feature-complete as an MVP. Core product is done. Audio content, 
full exercise seeding, and real ASR for speech recognition are 
the remaining gaps before a production launch.
